ABOUT our GIS

This project was built in a team of three for a 2nd year course ECE244.

Programming language used: C++. It was divided into 4 major milestones

1) Building efficient data structures using the OpenStreetMap API

2) Display map such as road/points of interest/highways such that user can interact with it

3) Build shortest path algorithm between two points on the map

4) Extend the shortest path algorithm if there are several randomized stops between source and destination
